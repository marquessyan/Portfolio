Aplicação Web de Previsão do Tempo com Geolocalização

Projeto desenvolvido com foco em fornecer informações climáticas em tempo real, utilizando as APIs OpenWeather e WeatherAPI. A aplicação exibe dados sobre o clima atual e previsão para os próximos dias (limite de até 3 dias, conforme restrição da API).

Principais Funcionalidades:
  • Detecção automática de localização via geolocalização do navegador;
  • Busca por condições climáticas em qualquer cidade do mundo;
  • Interface responsiva e intuitiva;
  • Backgrounds dinâmicos que variam conforme a temperatura atual, proporcionando uma experiência visual imersiva.

Tecnologias utilizadas: HTML, CSS, JavaScript, APIs REST (OpenWeather, WeatherAPI), Geolocation API.

Este projeto demonstra habilidades em consumo de APIs externas, manipulação de dados JSON, integração com geolocalização e desenvolvimento de interfaces dinâmicas com foco na experiência do usuário.
